# AppLinkCodeLab

This project demonstrates Android App Links.

To enable App Links for `carpe4me.github.io`, host the file `.well-known/assetlinks.json` from this repository on that domain. The file includes the SHA-256 signature `1E:99:0E:47:FD:28:4E:AE:A2:50:F2:0A:C6:E0:F1:9B:8B:E8:F8:B6:A3:73:D9:48:63:60:55:1F:02:68:6D:D2`.
